# 這裡只是 git 練習

## 作業一

1. Clone 原本的 repository。

## 作業二

1. Add 一個新的檔案。
2. Commit!

## 作業三

1. 製作一個新的 branch。
2. Checkout 到這個新的 branch。
3. Push 到 GitHub。

## 作業四 (下週再來!)

1. Create a new branch named to "merge_test_<your SID>"
2. Update the file: introduction.txt.
3. Commit the file
4. Merge to master
